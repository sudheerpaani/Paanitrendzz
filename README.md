**account.html** 
file in the Paanitrendzz repository is an interactive HTML page that forms the "My Account" section of the Paanitrendzz website. Here’s a summary of its structure and features:

General Overview:
Title: "My Account – Paanitrendzz".
Design: A clean, responsive layout with styled elements for navigation, content presentation, and interactivity. It uses custom CSS with a focus on aesthetics (e.g., gradients, modern fonts).
Interactive Features: Includes sections for user data, navigation, and JavaScript-driven actions for a dynamic user experience.
Navigation:
Header:

Displays the brand's logo and navigation buttons for "Shop Now" and "Logout."
Includes a "User Chip" to show the user’s avatar and name.
Sidebar:

Contains navigation links to different sections:
Overview
Orders
Wishlist
Profile
Addresses
Refer & Earn
Settings
Logout option.
Content Sections ("Panels"):
Overview:

Shows user account statistics (e.g., total orders, delivered, spending).
Displays recent orders with tracking and actions (e.g., review, return).
Orders:

Allows the user to track, return, or review past orders.
Includes a filter to display orders by statuses like delivered or pending.
Wishlist:

Displays favorite items with options to remove or add to the cart.
Profile:

Manages personal information like name, phone, and email.
Allows the user to change their password and notification preferences.
Addresses:

Displays and manages saved shipping addresses.
Includes default markings and options to edit or set an address as default.
Refer & Earn:

Contains referral code sharing options (e.g., WhatsApp, clipboard copy) with rewards.
Shows referral statistics (e.g., cashback earned).
Settings:

Allows toggling privacy settings like profile visibility and data sharing.
Contains a language selection dropdown and account deletion (via support).
Interactivity:
JavaScript Functions:

Handles session loading, panel navigation, and profile updates.
Controls actions like copying a referral code, order filtering, and notifications.
Displays success or error messages with a toast system.
Responsiveness:

Includes media queries for mobile-friendly design.
This file defines a well-structured, modular, and highly interactive user account page for the Paanitrendzz platform.
